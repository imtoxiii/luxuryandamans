# Home Page Image Guide

## Folder Structure

```
home/
├── hero/           # Hero section slideshow images
├── destinations/   # Destination card images
├── experiences/    # Experience card images
└── testimonials/   # Testimonial/review images
```

## 1. Hero Images (`hero/` folder)

**Purpose**: Main slideshow on homepage hero section

**Naming Convention**:
- `1.jpg`, `2.jpg`, `3.jpg`, etc.

**Specifications**:
- **Resolution**: 1920x1080px or higher
- **Aspect Ratio**: 16:9
- **Quantity**: 5-10 images
- **File Size**: Under 500KB each

**Content Ideas**:
- Stunning beach landscapes
- Aerial views of islands
- Water activities
- Sunset/sunrise scenes
- Tourist attractions

---

## 2. Destination Images (`destinations/` folder)

**Purpose**: Cards in the "Popular Destinations" section

**Naming Convention**:
- Use destination slugs: `port-blair.jpg`, `havelock-island.jpg`, `neil-island.jpg`, etc.
- Or numeric: `1.jpg`, `2.jpg`, `3.jpg`, etc.

**Specifications**:
- **Resolution**: 800x600px minimum
- **Aspect Ratio**: 4:3 or 16:9
- **Quantity**: 6-12 images
- **File Size**: Under 300KB each

**Destinations to Cover**:
- Port Blair
- Havelock Island (Swaraj Dweep)
- Neil Island (Shaheed Dweep)
- Ross Island
- Baratang Island
- Diglipur
- Long Island
- North Bay Island
- Jolly Buoy Island
- Red Skin Island

---

## 3. Experience Images (`experiences/` folder)

**Purpose**: Cards in the "Experiences" section

**Specifications**:
- **Resolution**: 800x600px minimum
- **Aspect Ratio**: 4:3 or 1:1
- **Quantity**: 8-15 images
- **File Size**: Under 300KB each

- Scuba Diving
- Snorkeling

- Island Hopping

---

## 4. Testimonial Images (`testimonials/` folder)

**Purpose**: Customer photos for testimonials/reviews

**Naming Convention**:
- Customer names: `john-doe.jpg`, `sarah-smith.jpg`
- Or numeric: `1.jpg`, `2.jpg`, `3.jpg`, etc.

**Specifications**:
- **Resolution**: 400x400px (square)
- **Aspect Ratio**: 1:1
- **Quantity**: 10-20 images
- **File Size**: Under 100KB each

**Content Ideas**:
- Happy customer photos
- Couples/families enjoying
- Professional headshots
- Candid vacation shots

---

## Image Optimization Tips

1. **Compression**: Use tools like TinyPNG, ImageOptim, or Squoosh
2. **Format**: JPG for photos, PNG for graphics with transparency
3. **Naming**: Use lowercase, hyphens for spaces, no special characters
4. **Quality**: Balance between file size and visual quality (80-85% quality)
5. **Backup**: Keep original high-res versions separately

---

## Automatic Loading

The system will automatically:
✓ Load all images from each folder
✓ Display them in the appropriate sections
✓ Handle missing images gracefully
✓ Optimize loading performance

**Simply drop your images into the folders and they'll appear on the website!**

## Quick Checklist

Before adding images:
- [ ] Images are properly compressed (under 500KB)
- [ ] Images are properly named (lowercase, hyphens)
- [ ] Images match the recommended resolutions
- [ ] Images are in supported formats (JPG, PNG, WebP)
- [ ] Images are relevant to the content
- [ ] Images are high quality and professional

---

## Need Help?

If images don't appear:
1. Check the browser console for errors
2. Verify file names match the expected pattern
3. Ensure images are in the correct folder
4. Clear browser cache and refresh
5. Check image file formats (should be .jpg, .jpeg, .png, or .webp)
