# Package Image Guide

## How to Add Images

Simply drop your images into the subfolders with numeric names:

### 1. Hero Images (Slideshow)
- **Folder**: hero/
- **Naming**: 1.jpg, 2.jpg, 3.jpg, etc.
- **Usage**: Main slideshow on package detail page
- **Recommended**: 5-8 images

### 2. Gallery Images
- **Folder**: gallery/
- **Naming**: 1.jpg, 2.jpg, 3.jpg, etc.
- **Usage**: Additional images throughout the page
- **Recommended**: 10-20 images

### 3. Highlight Images
- **Folder**: highlights/
- **Naming**: Based on highlight names (e.g., scuba-diving.jpg)
- **Usage**: Specific images for each package highlight
- **Recommended**: 1 per highlight (4-6 total)

### 4. Hotel Images
- **Folder**: hotels/
- **Naming**: 1.jpg, 2.jpg, 3.jpg, etc.
- **Usage**: Hotel and accommodation photos
- **Recommended**: 3-5 images per hotel

## Image Specifications

- **Format**: JPG, JPEG, PNG, or WebP
- **Resolution**: Minimum 1920x1080px for hero images
- **File Size**: Keep under 500KB (use compression)
- **Aspect Ratio**: 16:9 preferred for hero, flexible for others

## Automatic Detection

The system automatically:
 Loads all hero images for the slideshow
 Uses the first hero image for package cards
 Falls back to default images if none found
 Displays images in numeric order

**Just drop your images and refresh the page!**
